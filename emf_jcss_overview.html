電磁流量計と実流量校正設備の全体像

東京計器 実流量校正設備（JCSS）の構成を例に、検出器・変換器・調節計・オペレーション設備を整理

本資料では、東京計器株式会社の実流量校正設備（公式紹介ページ）を例に、 電磁流量計の検出器・変換器と、それを取り巻く調節計（制御コンピュータ）およびオペレーション設備の関係を、社会人1年目にも分かるように解説します。

ポイント：「水の流れ」→「電気信号」→「自動制御」→「人の操作」という流れで見ると、全体像がつかみやすくなります。

1. 実流量校正設備の全体イメージ

実流量校正設備は、大きく分けて次のような設備で構成されています。

  • 高い位置から水を安定して供給する高架水槽
  • 実際に水を流す管路試験ライン・開水路試験ライン
  • 基準となるマスターメータ（標準流量計）と秤量システム
  • ポンプ、制御弁、回転式ダイバータなどの操作部（設備を実際に動かす部分）
  • 流量を監視・制御する制御コンピュータ／調節計とコントロールルーム

2. 電磁流量計を含む計装ループの全体ブロック図

実流量校正設備の中で、電磁流量計と周辺設備の関係を一つの計装ループとして見ると、次のようになります。

これを「何がどこからどこへ伝わるか」で見ると、次のような流れになります。

  • 水の流れが、被試験電磁流量計の検出器で検知される
  • 検出された信号が変換器で電気信号（例：4–20 mA）に変えられる
  • その信号を調節計（制御コンピュータ）が受け取り、「もっと流す／絞る」を計算する
  • 計算結果が操作部（ポンプ・制御弁など）に送られ、実際の流量が変わる
  • 全体の状態をオペレータがコントロールルームで監視・操作する

3. 電磁流量計の「検出器」と「変換器」

3-1 検出器（センサ部）

電磁流量計の検出器は、配管に直結して取り付けられる流量計本体です。内部には次のような構造があります。

  • 内側にライニング（ゴムや樹脂）を施した計測管
  • 管の外側に巻かれた励磁コイル（磁石の役割）
  • 管内の両側に取り付けられた電極（水の中の電圧を拾う）

水がこの計測管を流れると、磁界と水の流れの組み合わせで小さな電圧が発生します。
この電圧の大きさが流量に比例しているため、電圧を測れば流量が分かります。

イメージとしては、「水道のメーターの中で、電気的に水の流れを感じている部分」が検出器です。

3-2 変換器（トランスミッタ）

電磁流量計の変換器は、検出器から来た微小な電圧信号を、 制御盤で扱える信号に変換する「頭脳＋信号変換器」です。通常、次のような役割を持ちます。

  • 検出器からの微小な電圧を増幅し、ノイズを取り除く
  • 流量に換算して表示したり、4–20 mA やパルス信号として出力する
  • ゼロ点調整、スパン設定、単位（m3/h など）の設定
  • 自己診断（電極のコーティング、配線異常など）の監視

実流量校正設備では、被試験の電磁流量計の変換器から出た信号を、 比較用のマスターメータや秤量システムの結果と見比べて、 「どの程度ずれているか」を評価します。

信号の向き 内容 典型的な信号種別
検出器 → 変換器 電極で検出した微小電圧 mV レベルのアナログ信号
変換器 → 調節計 瞬時流量、積算流量 4–20 mA（AI）、パルス、デジタル通信

4. 設備的に考える「調節計（動作）」

実流量校正設備では、コントロールルーム内の制御コンピュータ／PLCが、 実質的に調節計（コントローラ）の役割を果たします。

4-1 調節計の基本動作

調節計は、次のような流れで動作します（流量制御の例）。

  1. オペレータが目標流量（設定値）を入力する（例：Q = 500 m³/h）。
  2. 調節計が、
    • 被試験電磁流量計の信号
    • マスターメータ（標準流量計）の信号
    を取り込み、実際の流量（測定値）を把握する。
  3. 設定値と測定値の差（偏差）から、「どれだけバルブを開ける／閉じるか」をPID制御などで計算する。
  4. 計算結果を操作信号（例：4–20 mA の開度指令）として、制御弁やポンプに出力する。
  5. その結果、実際の流量が変わり、測定値が再び調節計に戻ってくる（制御ループ）。

4-2 校正時のイメージ（簡略）

電磁流量計を校正するときは、次のような流れで調節計が働いているとイメージできます。

  • 指定した流量になるように、ポンプの運転や制御弁の開度を自動調整する
  • 流量が安定したら、秤量システムやマスターメータと連携して計測タイミングを制御する
  • 必要な複数点（例：25%、50%、75%、100%）の流量条件を、自動で順番に作り出す

設備的に見れば、 「流量条件を作る頭脳」＝調節計／制御コンピュータ と考えると分かりやすくなります。

5. オペレーション設備と人の操作

オペレーション設備は、主にコントロールルーム側と現場側に分けて考えると整理しやすくなります。

5-1 コントロールルーム側のオペレーション

コントロールルームには、システムを操作するための操作盤・監視画面がまとめられています。 ここでオペレータが行う代表的な操作は次の通りです。

  • どの試験ラインを使うかを選択（口径 50A ～ 600A、開水路ライン など）
  • 試験方法の選択：
    • マスターメータによる比較法
    • 秤量システムによる秤量法
  • 目標流量値の設定（例：200, 500, 1000 m³/h）
  • 試験開始／停止ボタンの操作
  • 警報の確認と、必要に応じた復旧操作
  • 試験結果（被試験電磁流量計と標準の差）の確認・保存・印刷
現場でいうと、「DCSのオペステーション」や「中央監視盤」と同じイメージで考えると理解しやすい部分です。

5-2 現場側（フィールド）の操作

現場（管路試験ラインまわり）では、次のような操作や点検が行われます。

  • 被試験の電磁流量計・配管の取り付け／取り外し
  • フランジ接続部やパッキンの漏れ点検
  • 電磁流量計の電極部が水で満たされているかの確認
  • 必要に応じたバルブの開閉（準備時の手動操作）
  • ポンプや回転式ダイバータの外観異常の確認

これらの現場作業と、コントロールルームの自動制御が組み合わさって、 安全かつ正確な流量校正が行われます。

6. この設備の中での「検出部・変換部・調節部・操作部」対応まとめ

最後に、工業計器の4要素と、実流量校正設備のどの部分が対応するかを整理します。
