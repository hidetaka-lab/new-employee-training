計装とは何か

「測る」「伝える」「制御する」の基本概念

〜 新入社員向け技術説明資料 〜

1. 計装（Instrumentation）の定義

計装とは：プラントや設備の運転状態を測定し、その情報を伝送して、最適な状態に制御するための技術体系です。

英語の「Instrumentation」は「計器を装備すること」を意味し、日本では「計測」と「装置」を組み合わせて「計装」と呼ばれています。

上下水道施設や製造プラントにおいて、計装は設備の「目」「神経」「頭脳」として機能し、安全で効率的な運転を実現する重要な役割を担っています。

2. 計装の3つの基本概念

📊
測る（Measure）
温度、圧力、流量、液位などのプロセス変数を正確に測定し、数値化します。現場の状態を把握するための第一歩です。
📡
伝える（Transmit）
測定したデータを電気信号（4-20mA、1-5V等）やデジタル通信で制御室まで伝送します。情報の橋渡し役です。
⚙️
制御する（Control）
収集したデータを基に、バルブやポンプなどの機器を自動制御し、プロセスを最適な状態に保ちます。

4. 主要な計装機器

4.1 測定機器（センサー・発信器）

圧力計・圧力発信器
配管内の圧力を測定。ポンプの吐出圧力監視などに使用。
流量計
電磁式、超音波式など。上水の配水量や薬品注入量の測定。
液位計
タンクや池の水位測定。投込式、超音波式、レーダー式など。
温度計
測温抵抗体（Pt100）、熱電対など。プロセス温度の監視。
水質計器
pH計、濁度計、残留塩素計など。水質管理に必須。
分析計
溶存酸素計、導電率計など。プロセス管理用。

4.2 制御機器

PLC（プログラマブルロジックコントローラ）
シーケンス制御の中核。入出力信号を処理し、機器を自動制御。
DCS（分散制御システム）
大規模プラント向け。高度なプロセス制御を実現。
調節計
PID制御などで目標値に自動調整。流量や圧力の制御に使用。
SCADA
監視制御システム。広域の設備を中央監視室から一元管理。

4.3 操作端（アクチュエータ）

電動弁・空気作動弁
流量調整や開閉制御。制御信号により自動操作。
インバータ
ポンプやファンの回転数制御。省エネ運転を実現。
電磁弁
小口径配管の開閉制御。薬品注入ラインなどで使用。

5. 信号の種類（I/O）

AI
Analog Input
アナログ入力信号
4-20mA、1-5V
例：流量、圧力、温度
AO
Analog Output
アナログ出力信号
4-20mA、1-5V
例：バルブ開度指令
DI
Digital Input
デジタル入力信号
ON/OFF、開/閉
例：運転/停止状態
DO
Digital Output
デジタル出力信号
ON/OFF指令
例：ポンプ起動/停止
標準信号について：
  • 4-20mA：工業標準の電流信号。ノイズに強く長距離伝送が可能
  • 1-5V：電圧信号。PLCの入力カードで直接受信可能
  • 接点信号：無電圧接点（ドライ接点）または有電圧接点

6. 計装工事の実務

6.1 計装工事の流れ

  • 設計段階：P&ID（配管計装図）の作成、計装品の選定、制御ロジック設計
  • 施工段階：計器取付、配線工事、盤製作・設置、ケーブル敷設
  • 試験段階：ループチェック、模擬信号試験、インターロック試験
  • 試運転段階：実液試験、制御調整（PIDチューニング）、総合動作確認

6.2 ループチェックとは

ループチェック：計装ループ（センサー → 伝送器 → PLC → 操作端）の一連の動作を確認する重要な試験です。
  • 模擬信号を入力し、正しく伝送・表示されるか確認
  • 制御出力が正しく動作するか確認
  • 警報設定値で適切に警報が発報するか確認

6.3 計装盤の種類

  • 現場操作盤：現場での機器操作用。防水・防塵構造
  • 中継端子盤：現場信号の中継・分岐用
  • 動力制御盤：ポンプ等の動力機器制御用
  • 計装盤：PLC収納、信号処理用
  • 監視操作盤：中央監視室での監視・操作用

7. 上下水道施設での計装の役割

7.1 浄水場での計装

  • 取水施設：河川水位監視、取水ゲート制御
  • 凝集沈殿：濁度測定、薬品注入制御、攪拌機制御
  • ろ過池：ろ過水頭監視、逆洗制御
  • 消毒：塩素注入制御、残留塩素監視
  • 送配水：配水池水位監視、送水ポンプ制御、配水圧力制御

7.2 下水処理場での計装

  • 流入施設：流入量測定、ゲート・ポンプ制御
  • 水処理：DO（溶存酸素）制御、MLSS監視、返送汚泥制御
  • 汚泥処理：汚泥濃度測定、脱水機制御、消化ガス監視
  • 放流：水質監視（BOD、SS、大腸菌等）
⚠️ 安全管理の重要性：
  • 硫化水素などの有毒ガス検知システムの設置
  • 緊急遮断弁のインターロック制御
  • 停電時の非常用電源への自動切替
  • 異常時の警報発報と自動停止機能

8. 最新の計装技術トレンド

8.1 IoT・クラウド連携

  • 無線センサーネットワークによる広域監視
  • クラウドSCADAによる遠隔監視・制御
  • ビッグデータ解析による予知保全

8.2 AI・機械学習の活用

  • 運転最適化（薬品注入量の最適制御等）
  • 異常検知・故障予測
  • エネルギー管理の最適化

8.3 サイバーセキュリティ

  • 制御システムのセキュリティ対策強化
  • ネットワーク分離、ファイアウォール設置
  • 定期的なセキュリティ監査の実施

9. 計装エンジニアとして身につけるべきスキル

基礎知識
  • 電気・電子工学の基礎（オームの法則、電子回路）
  • 制御理論（PID制御、シーケンス制御）
  • 流体力学・熱力学の基礎
  • プロセス工学の理解
実務スキル
  • P&ID（配管計装図）の読解・作成能力
  • PLCプログラミング（ラダー図、ファンクションブロック）
  • HMI/SCADA画面の設計・構築
  • 計装ループの設計・試験・調整
  • トラブルシューティング能力
コミュニケーション能力
  • 顧客（施設運転員）との調整能力
  • 他工種（電気・機械・土木）との連携
  • 技術文書の作成能力
  • プレゼンテーション能力

10. まとめ

計装は「測る」「伝える」「制御する」という3つの基本概念を通じて、プラント設備の安全で効率的な運転を支える重要な技術です。

計装エンジニアの使命：

  • 24時間365日、安定した設備運転の実現
  • 省エネルギー・省力化の推進
  • 安全性の確保と事故の未然防止
  • 環境規制への適合と水質の確保
